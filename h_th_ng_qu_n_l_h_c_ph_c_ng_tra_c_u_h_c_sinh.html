Cấu hình kết nối hệ thống thực tế (Trọn Đời)

Hướng dẫn cài đặt thực tế:

  1. Mở một file Google Sheets mới.
  2. Vào Tiện ích mở rộng (Extensions) -> Apps Script. Dán toàn bộ code tệp backend.js vào và lưu lại.
  3. Nhấn nút Triển khai (Deploy) -> Triển khai mới (New deployment). Chọn loại là Ứng dụng web (Web app).
  4. Thiết lập: Người thực thi là Tôi (Me), và Người có quyền truy cập là Bất kỳ ai (Anyone). Tiến hành triển khai và sao chép đường dẫn Web App nhận được dán vào ô trên!
  5. Đăng ký tài khoản miễn phí trên SePay, Casso hoặc PayOS, và dán URL Web App trên làm Webhook của bạn.
